R Programming

This is an article on R programming language

An Introduction
R is an integrated suite for data manipulation, calculation and its graphical representation. It also provides other tools to handle data and its storage.
It's a newer implementation of S language.
It's an environment in which many classic and modern statistical techniques have been implemented and hence used as a statistics system by many. Technically R is a case sensitive, expression language where a statistical analysis is done in a series of steps with intermediate result being stored in objects.

R Command Prompt
R provides you with a prompt where a user can execute commands. Elementary commands consist of either expressions or assignments.

  • If an expression is given as a command, it is evaluated, printed (unless specifically made invisible), and the value is lost.
  • An assignment also evaluates an expression and passes the value to a variable but the result is not automatically printed.
Commands can be separated either by a semi-colon (;), or by a newline. Elementary commands can be grouped together into one compound expression by braces ({ and }).
Comments can be put almost anywhere, starting with a hash mark (#), everything to the end of the line is a comment.
The entities that R creates and manipulates are known as objects.
The R command
    > objects()
or ls() can be used to display the names of the objects which are currently stored within R.
The collection of objects currently stored is called the workspace.
To remove objects the function rm is available:
    > rm(x, y, z, ink, junk, temp, foo, bar)
To quit the R program the command is
    > q()
